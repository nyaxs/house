# 总论
- 我们的定位是，首次置业型的刚需客户，附带一点保值的需求
- 我们的目标是使用50-60w的首付预算，在苏州买到总价166-200w的房子
- 挑选房子的关键指标是：
  - 价格
  - 地铁（交通、通勤）
  - 宜居（面积、环境、户型、商业）
  - 学区
  - 产业和城市规划
- 二手房还是新房
  - 二手房
    - 优点
    - 缺点
  - 新房
    - 优点
    - 缺点
